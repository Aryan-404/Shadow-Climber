About The Project

Shadow Climber is a multi-level HTML5 platformer built using JavaScript and Canvas API. The game is containerized with Docker and deployed on AWS EC2 provisioned using Terraform.

This project demonstrates Infrastructure as Code (IaC), containerized deployment, and cloud-native hosting practices.
